Comments to CRAN maintainers for submission
===========================================

## Version 0.9-8

This is a minor maintenance release addressing two issues as requested by the CRAN maintainers:

 1. due to changes in *vegan* package (version 2.7.1), the reference output for examples no longer matched the output generated by the package, and
 2. there was a single escaped & in `man/jackal.Rd`.

Both issues are rectified in the 0.9-8 release.

## Test Environments

* Linux, Ubuntu (24.04) (r-oldrelease, r-release & r-devel)
* MacOS X (r-release, locally and on GitHub & Mac-builder)
* win-builder (r-oldrelease, r-release & r-devel)
